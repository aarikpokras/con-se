What do we filter out?

When you search something, for example, "what is a water bottle?", the following are filtered out of your search:

So in other words, we filter out "what is", "who is", "a", "an", spaces, punctuation, and more just to get what you're searching for.
So you're just left with "waterbottle". More on that later.

So how does it actually work?

Well, we use the JavaScript fetch API to get JSON (JavaScript Object Notation) files in the /res directory that contain basic information about the site. For example: 
{
  "name": "Concise Search Engine",
  "description": "It's a search engine.",
  "url": "https://con-se.glitch.me/"
}
Anyway, "waterbottle" from earlier. So now that we have that, we look for files in the /res folder that match waterbottle.json. If we find it, we display its JSON contents to you. If not, that's how you get the "no results" page.
